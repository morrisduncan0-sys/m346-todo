# M346 LB3 – To-Do App (Coolify)

Einfache To-Do App mit Node.js und PostgreSQL, deployed via Coolify.

## Zusatz-Elemente
- **Element 1 – Webhook**: GitHub Push → automatisches Deployment in Coolify
- **Element 2 – Datenbank**: PostgreSQL-Service in Coolify, verbunden via `DATABASE_URL`
- **Element 3 – Health-Check**: `/health` Endpoint prüft DB-Verbindung
